(async) getHistorico(mineralName, periodoopt) → {Promise.<({mineral: string, periodoSeleccionado: string, precioActual: (number|null), unidad: string, historico: Array.<{fecha: string, precio: number}>}|null)>}

Devuelve el histórico de precios de un mineral para el período seleccionado. Intervalos: 30 d → diario; 12 m / 5 y → mensual (para no saturar la gráfica). Incluye también el precio actual en tiempo real como `precioActual`.
