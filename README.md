## Datos para la conexión a la base de datos

- *Nombre de la base de datos:* minex
- *Usuario:* root
- *Contraseña:* test
- *Host:* localhost
- *Puerto:* 3306

## Ejecución del Proyecto

### Backend

1.  Navegar al directorio del backend:
    bash
    cd ../minex-backend
    
2.  Instalar dependencias:
    bash
    npm install
    
3.  Iniciar el servidor de desarrollo:
    bash
    npm run dev
